# Loja do Seu Manoel

Este projeto é uma API .NET Core que empacota pedidos otimizando o uso de caixas.
Não foi possível adicionar todos os requisitos do projeto, como docker e sql, mas mesmo assim resolvi enviar o projeto
para demonstrar minhas habilidades no código, o tempo foi curto mas dei meu máximo.

## Funcionalidades

- Recebe pedidos com produtos
- Calcula quais caixas usar
- Retorna empacotamento otimizado

## Como usar

1. Enviar uma requisição POST para `/api/empacotamento/empacotar`
2. Passar a lista de pedidos no corpo da requisição
